# adv-os-project1
Project 1 from UT Advanced Operating Systems MSCS course.

Warmup exercises to become familiar with C programming.

Project contains:
* [reverse.c](reverse/reverse.c): prints input in reverse by using a linked list
* [wcat](utilities/wcat/wcat.c): simplified version of `cat`
* [wgrep](utilities/wgrep/wgrep.c): simplified version of `grep`
* [wzip](utilities/wzip/wzip.c)/[wunzip](utilities/wunzip/wunzip.c): simplified version of `zip`/`unzip` that uses [Run-length Encoding](https://en.wikipedia.org/wiki/Run-length_encoding) compressions
